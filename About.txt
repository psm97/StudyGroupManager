B-1. float 스케일 불일치 (가장 광범위한 문제)

필드: risk_score
DB 저장값: 0.0~1.0
프론트엔드 기대값: 0~100 정수
파손되는 UI: 게이지 SVG 원형 (stroke-dasharray = score × 2.76) — 값이 0.12이면
  원의 1.2%만 채워짐
────────────────────────────────────────
필드: dropout_probability
DB 저장값: 0.0~1.0
프론트엔드 기대값: 0~100 %
파손되는 UI: 탈퇴 예측 지표 게이지 동일
────────────────────────────────────────
필드: achievement_probability
DB 저장값: 0.0~1.0
프론트엔드 기대값: 0~100
파손되는 UI: 플래너 Arc 게이지 (score × 3.64)
────────────────────────────────────────
필드: performance_score
DB 저장값: 0.0~1.0
프론트엔드 기대값: 0~100
파손되는 UI: 주차별 성과 차트 (Y축 min:0, max:100)

뷰에서 반드시 × 100 변환 후 응답해야 함.

const LEADER_GROUP_IDS = new Set([1]); // 현재 사용자가 리더인 그룹 ID
